Integration Notes

Where to plug in endpoints later.

  1. Open script.js and set CONFIG.MOCK = false.
  2. Fill in CONFIG.ENDPOINTS for upload, stream, and chat.
  3. Match backend request field names (file key, session/video id key).
  4. If streaming is MJPEG, we’ll use an <img> tag; if it’s MP4/HLS, we’ll use the <video> player.
CONFIG = {
  MOCK: false,
  BASE_URL: "http://localhost:8000",
  ENDPOINTS: { upload: "/upload", chat: "/chat", stream: (id) => `/stream/${id}` }
};
